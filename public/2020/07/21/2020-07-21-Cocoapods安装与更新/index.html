4.升级ruby

查看ruby版本：ruby -v

安装homebrew

可以使用rvm和brew来安装最新的ruby

  • rvm:

    rvm install 2.5.3

    rvm use 2.5.3 --default

  • bew:

    brew install ruby

5.安装cocoapods

sudo gem install -n /usr/local/bin cocoapods
